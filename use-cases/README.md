# Rise.ai Integration Use Cases

This directory contains examples of how to integrate Rise.ai with your ecommerce store in different scenarios.

## Available Use Cases

*   [Custom Checkout](./custom-checkout/): Demonstrates how to build a custom checkout flow that incorporates Rise.ai gift cards and promotions.
*   [Native Gift Cards Sync](./native-gift-cards-sync/): Shows how to synchronize your store's native gift card system with Rise.ai.
*   [Shopify Headless Store](./shopify-headless-store/): Provides an example of integrating Rise.ai into a headless Shopify store built with Hydrogen.

Each use case has its own `README.md` with specific instructions on how to set up and run the example.
