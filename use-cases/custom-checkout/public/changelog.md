# CHANGELOG

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2024-01-XX

### Added
- Initial implementation of custom checkout flow with Rise.ai gift card integration
- Gift card query, decrease, and increase API integration
- Fraud prevention checks (no discounts, no gift card products in cart)
- Balance validation
- Payment processing with automatic gift card void on failure
- React components for gift card input and checkout form
- Comprehensive error handling
- Success page for order confirmation
